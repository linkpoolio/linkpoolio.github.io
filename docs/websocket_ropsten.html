Ropsten WebSocket

We offer a public Ethereum RPC for main (Foundation) that provides access to the WebSocket RPC.

Endpoint

WebSocket RPC: https://ropsten-rpc.linkpool.io/ws

Example Usage

wscat -c wss://ropsten-rpc.linkpool.io/ws
connected (press CTRL+C to quit)

Limitations

The RPC is behind a web application firewall that restricts traffic based on the rate of calls.

Rate Limiter

The RPC is limited to 2,000 calls per 5 minutes. If that is exceeded, then the source IP address is blocked.
